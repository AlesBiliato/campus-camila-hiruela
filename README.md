# Campus Camila Hiruela

Web multipágina desarrollada para el Campus Camila Hiruela, un campus de vóley en Jujuy. El proyecto está construido con HTML, JavaScript, Sass y Vite, con un enfoque en diseño visual, responsive y experiencia de usuario.

## Demo

[Ver sitio publicado](https://campus-camila-hiruela.netlify.app/)

## Tecnologías

- HTML5
- JavaScript
- Sass
- Vite
- Netlify

## Secciones principales

- Inicio con vídeo de portada y cuenta atrás
- Quiénes somos
- Información del campus
- Página 404 personalizada

## Cómo ejecutar el proyecto en local

```bash
npm install
npm run dev
```

## Generar versión de producción

```bash
npm run build
```

La build final se genera en la carpeta `dist`.
